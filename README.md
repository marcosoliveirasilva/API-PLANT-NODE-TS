# Projeto API NodeJS (API-PLANT-NODE-TS)

Projeto API NodeJS

## Install the dependencies
```bash
npm
# or
yarn install
```

### create and populate the database
```bash
yarn knex:migrate
yarn knex:seed
```

### Start the app in development mode
```bash
yarn start
```
